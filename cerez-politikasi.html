Çerez Politikası

TEREA ILUMA TÜRKİYE ("Biz") olarak, web sitemizi ziyaretleriniz sırasında deneyiminizi geliştirmek ve size daha iyi hizmet sunmak amacıyla çerezler (cookies) kullanmaktayız. Bu Çerez Politikası, web sitemizde kullandığımız çerez türlerini, kullanım amaçlarını ve çerez tercihlerinizi nasıl yönetebileceğinizi açıklamaktadır.

1. Çerez Nedir?

Çerezler, bir web sitesini ziyaret ettiğinizde cihazınıza (bilgisayar, tablet, akıllı telefon vb.) kaydedilen küçük metin dosyalarıdır. Bu dosyalar, web sitesinin sizi hatırlamasını, tercihlerinizi kaydetmesini veya site içi gezinme bilgilerinizi saklamasını sağlar.

2. Hangi Çerezleri Kullanıyoruz ve Neden?

Web sitemizde farklı türde çerezler kullanmaktayız:

  • Zorunlu Çerezler: Web sitemizin temel işlevleri için gerekli olan çerezlerdir. Örn: Oturum çerezleri, site navigasyonu. Bu çerezler olmadan web sitesi düzgün çalışamaz.
  • Performans/Analitik Çerezler: Web sitemizin nasıl kullanıldığını analiz etmek, ziyaretçi sayılarını ve popüler sayfaları belirlemek amacıyla kullanılır. Google Analytics çerezleri bu kategoriye girer. Bu çerezler, sitenin performansını iyileştirmemize yardımcı olur ve doğrudan sizi tanımlamaz.
  • İşlevsel Çerezler: Web sitesinin daha kişiselleştirilmiş bir deneyim sunmasına olanak tanır. Örn: Dil tercihleri, bölge ayarları.
  • Pazarlama/Hedefleme Çerezleri: İlgi alanlarınıza daha uygun reklamlar sunmak ve reklam kampanyalarının etkinliğini ölçmek için kullanılır. Bu çerezler genellikle üçüncü taraf reklam ağları tarafından yerleştirilir. (Şu anki sitenizde doğrudan reklam çerezleri görünmemektedir, ancak gelecekte eklenebilecek durumlara karşı bir placeholder'dır).

3. Çerezleri Yönetme

Tarayıcınızın ayarlarını değiştirerek çerezleri kabul edebilir, reddedebilir veya silme uyarısı alabilirsiniz. Çerez ayarlarınızı yönetmek için tarayıcınızın yardım menüsüne bakabilir veya aşağıdaki bağlantılardan popüler tarayıcılar için bilgi edinebilirsiniz:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge

Çerezleri devre dışı bırakmanız durumunda web sitemizin bazı özelliklerinin tam olarak çalışmayabileceğini lütfen unutmayınız.

4. Bu Politikanın Değişiklikleri

Bu Çerez Politikası zaman zaman güncellenebilir. Herhangi bir değişiklik durumunda, güncellenmiş politikayı web sitemizde yayınlayacağız.
